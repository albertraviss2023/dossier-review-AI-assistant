# Benchmark Dataset Generator

This generator creates a synthetic eCTD/CTD-style benchmark dataset for pre-market authorization review:

```powershell
python synthetic_data/generate_dataset.py --config synthetic_data/dataset_config.yaml --num-dossiers 192 --output synthetic_dossier_dataset
```

Smoke run:

```powershell
python synthetic_data/generate_dataset.py --config synthetic_data/dataset_config.yaml --num-dossiers 16 --output synthetic_dossier_dataset_smoke
```

## Key outputs
- `labels/dossier_labels.jsonl`: dossier-level expected SOP outcomes.
- `labels/section_labels.jsonl`: section-level presence/completeness/extractability/correctness/consistency labels.
- `labels/chunk_labels.jsonl`: chunk-level text type and expected LLM action.
- `labels/issue_labels.jsonl`: controlled known query failures.
- `manifests/dossier_manifests.jsonl`, `page_manifests.jsonl`, `scan_manifests.jsonl`.
- `evaluation/expected_sop_assessments.jsonl` and `evaluation/dataset_validation_report.json`.

## Notes
- Query dossiers are generated by mutating a correct dossier in the same family.
- Split generation keeps family variants together to reduce leakage.
- Scan realism is captured through page/scan manifests and scan profiles for benchmark control.
